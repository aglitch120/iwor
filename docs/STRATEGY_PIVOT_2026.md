## 2. 新コアプロダクト: iwor Study

### 概要
FSRS（最新の間隔反復アルゴリズム）搭載のフラッシュカードアプリ。日本の医学教育に特化。

### 海外リサーチに基づく設計根拠
- Anki: 米国医学生の86.2%が使用、UIが2010年代のまま
- AnkiHub: 月$5、10万人以上。コミュニティデッキのリアルタイム更新が課金動機
- AMBOSS: 年$428、170カ国200万人。Qbank+Library+Anki統合
- 日本に存在しないもの: AnkiHub的コミュニティデッキ管理、AMBOSS的統合学習

### フリーミアム設計（AnkiHub/Brainscapeモデル準拠）

**FREE（永久無料）— 「一人で学ぶ」は無制限:**
- カード作成・編集: 無制限
- FSRS間隔反復: 無制限
- 自作デッキ数: 無制限
- 1日のレビュー枚数: 無制限
- デフォルトデッキ3つ（CBT基礎/国試必修/内科基礎）
- .apkgインポート
- 基本統計（今日の枚数、ストリーク）

**PRO（月あたり約¥550〜）— 「他人のデータ・AI・利便性」に課金:**
- コミュニティデッキ購読+リアルタイム更新同期
- AIカード生成（PDF/ノート→カード自動変換）
- 詳細統計・弱点分析・予測スコア
- ゲーミフィケーション（XP/リーグ）
- SNSシェア機能（Instagramストーリーズ等）
- 試験パーソナライズドスケジュール
- 奨学金制度（払えない人向け — AnkiHub同様）

### リスク評価（反証含む）
- 確信度70%: 日本の医学生にAnki的需要はあるが、QBカルチャーとの競合あり
- コミュニティデッキが自走する確信度: 40%（日本にAnki共有文化がない）
- 先行者利益: 50%（HOKUTOが参入すれば機能面で負ける可能性）
- **対策: 全ベットではなく段階的検証（後述の実装ロードマップ参照）**

## 6. LLM自動収集インフラ（学会スケジュール + 専門医単位 共通）

### アーキテクチャ
```
[月2回 cron trigger]
    → Cloudflare Worker: 43学会のURLリストをループ
    → fetch各ページ（HTML or PDF）
    → Claude Haiku APIで構造化抽出
    → KVに保存された前回データと比較
    → 差分あり → Slack/メール通知
    → 管理画面で承認 → KVを更新 → フロントに反映
```

### コスト
- 43学会 × 月2回 = 月86回のfetch + LLM呼び出し
- 月額約¥25（ほぼ無料）

### 法的リスク
- 事実情報の収集であり著作権侵害に該当しない
- 月2回のアクセス頻度は通常のブラウジングと区別不可
- 免責: 「各学会の公式サイトから情報を収集しています。誤りがある場合は公式サイトをご確認ください」
