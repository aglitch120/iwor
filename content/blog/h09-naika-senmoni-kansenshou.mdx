## 感染症領域の出題傾向

感染症領域は出題数が多い重要領域です。特に **抗菌薬の適切選択（原因菌別・患者背景別）** は毎年問われます。また、感染症のガイドライン改訂（敗血症のSurviving Sepsis Campaign等）も試験に反映されます。

## 敗血症の診断と管理（Sepsis-3）

### Sepsis-3診断基準

2016年のSepsis-3コンセンサスで定義が改訂されました。

**敗血症の定義**: 感染症に対する制御不能な宿主反応による生命を脅かす臓器障害
- 診断基準：感染症が疑われる＋**SOFAスコア ≥ 2点の急性増加**

**敗血症性ショックの定義**: 敗血症＋以下の2つを満たす
- 適切な輸液補充にもかかわらず平均動脈圧 **≥ 65mmHg** を維持するために昇圧薬が必要
- 血清乳酸値 **> 2mmol/L**（18mg/dL）

### Hour-1 Bundleの実行

敗血症では1時間以内の以下の行動が推奨されています。

1. **血液培養** 2セットを採取（抗菌薬投与前）
2. **乳酸値測定**（Lac > 2 mmol/Lなら再測定）
3. 広域抗菌薬の **即時投与**（1時間以内）
4. 循環不全には **晶質液30mL/kg** を3時間以内に投与
5. 輸液後も低血圧持続または Lac > 4mmol/L → 昇圧薬

## 肺炎の抗菌薬選択

### 市中肺炎（CAP）の治療

**重症度評価（A-DROPスコア、日本用）**
- Age（男性70歳以上、女性75歳以上）
- Dehydration（BUN ≥ 21mg/dL）
- Respiratory failure（SpO₂ ≤ 90%またはPaO₂ ≤ 60mmHg）
- Orientation（意識障害）
- Pressure（収縮期血圧 ≤ 90mmHg）

0〜1点：外来治療可、2点：入院、3点以上：重症入院、4〜5点：ICU

**外来・軽症**: アモキシシリンまたはマクロライド系（非定型肺炎疑いにはマクロライドまたはキノロン系）

**入院・中等症**: アンピシリン/スルバクタム＋マクロライド系、または抗菌薬スペクトラムの広いβラクタム＋マクロライド

**重症（ICU）**: 抗緑膿菌活性を持つβラクタム＋マクロライドまたはキノロン系

### 院内肺炎（HAP）・人工呼吸器関連肺炎（VAP）

原因菌として **緑膿菌・MRSA・Klebsiella** などの耐性菌を考慮した広域抗菌薬を使用。抗MRSA薬（バンコマイシン/リネゾリド）の追加が必要になることも多いです。

## HIV/AIDS の管理

### CD4値と日和見感染症

CD4値は免疫状態の指標で、低下すると特定の日和見感染症が発症するリスクが上がります。

| CD4値 | リスクのある日和見感染症 |
|-------|----------------------|
| < 500 | 帯状疱疹・細菌性肺炎 |
| < 200 | **ニューモシスチス肺炎（PCP）** — ST合剤で予防 |
| < 100 | **トキソプラズマ脳症** — ST合剤で予防 |
| < 50 | **CMV網膜炎** — ガンシクロビル、**MAC感染症** — アジスロマイシンで予防 |

### 抗HIV療法（cART）

現在のHIV治療は、3〜4剤を組み合わせた **cART（combination antiretroviral therapy）** が標準です。治療開始により HIV-RNAを「検出感度以下」まで抑制することが目標です。

## 感染性心内膜炎（IE）

### Duke修正診断基準

**大基準（2項目）**: 血液培養陽性（典型菌2セット以上）、心エコーで疣腫・弁穿孔

**小基準（5項目）**: 基礎心疾患/薬物静注、発熱≥38℃、血管症状（塞栓・出血）、免疫学的所見、血液培養陽性（大基準未達）

診断：大2、大1＋小3、小5 のいずれかで「確定」

### 治療の要点

- 原因菌はStreptococcus（最多）、Staphylococcus、Enterococcus
- 自然弁IE（ペニシリンG±アミノグリコシド）
- 人工弁IE（バンコマイシン＋リファンピシン±アミノグリコシド）
- 手術適応：弁機能不全・コントロール不能感染・大型疣腫

## まとめ

感染症領域の試験対策ポイント：

- **敗血症**: Sepsis-3定義（SOFAスコア）、Hour-1 Bundle（血培→抗菌薬→輸液）
- **肺炎**: A-DROPスコアで重症度評価、市中/院内で使用抗菌薬が異なる
- **HIV**: CD4値と日和見感染予防のカットオフを覚える
- **IE**: Duke修正診断基準の主要要素と抗菌薬治療期間

## よくある質問（FAQ）

### 抗菌薬の種類が多くて選択の基準がわかりません。どう覚えればいいですか？

「症候＋想定原因菌＋患者背景（アレルギー・腎機能・免疫状態）」で選ぶという原則を意識してください。市中肺炎の軽症→アモキシシリン/マクロライド、重症→広域βラクタム＋マクロライド、という代表的な選択肢から始めて、MRSA/緑膿菌リスクがある時は広域化する、という流れで覚えましょう。

### 敗血症のSOFAスコアを全部暗記する必要がありますか？

SOFAスコアの6臓器（肺・凝固・肝・循環・神経・腎）を知っていれば十分で、各スコアの閾値を全部暗記する必要はありません。「感染症＋急性臓器障害のある患者＝敗血症」という概念を理解し、Hour-1 Bundleの内容を確実に覚えることが優先です。

### HIV患者でCD4が200以下の時の予防投与は何を使いますか？

CD4 < 200でPCP（ニューモシスチス肺炎）予防にST合剤（トリメトプリム・スルファメトキサゾール）を使います。ST合剤はトキソプラズマ予防（CD4 < 100）にも使えるため、CD4 < 100になったら一石二鳥です。CMV（CD4 < 50）にはガンシクロビル、MACにはアジスロマイシン、と覚えてください。
