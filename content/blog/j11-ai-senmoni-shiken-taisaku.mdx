「ChatGPTを試験勉強に使いたいけど、どう活用すればいい？」「AIに問題を作ってもらうことはできる？」

内科専門医試験の対策にAIを活用することで、学習効率を大幅に向上できます。この記事では、ChatGPT・Claudeを使った実践的な試験対策の方法を解説します。

## AIを試験対策に使うメリット

### 1. 24時間いつでも質問できる

教科書を読んでいて「なぜ?」と思ったとき、すぐにAIに質問できます。指導医の都合を気にせず疑問を解消できるのは大きなメリットです。

### 2. 自分のレベルに合わせて説明してもらえる

「中級者向けに」「簡潔に」「症例ベースで」など、自分の理解度やニーズに合わせた説明を求めることができます。

### 3. 問題を無限に作れる

特定のテーマで問題を作成してもらうことで、市販の問題集にはない角度からの問題演習が可能になります。

## 実践的なAI活用プロンプト集

### 用途1: 問題を作ってもらう

```
【プロンプト例】
内科専門医試験向けに、心不全の治療に関するMCQ問題を5問作成してください。
難易度は中級程度で、選択肢は5択にしてください。
正答と解説も一緒に出してください。
```

**ポイント**: 「内科専門医試験レベル」と明示することで、適切な難易度の問題が作られます。

### 用途2: 苦手な疾患を深掘りする

```
【プロンプト例】
慢性腎臓病のステージ分類（G分類）と蛋白尿のA分類を、
臨床的な意味を含めてわかりやすく説明してください。
また、各ステージで特に注意すべき合併症と治療介入のポイントを教えてください。
```

### 用途3: 鑑別診断を練習する

```
【プロンプト例】
40代男性、突然の胸痛で救急来院。以下のバイタルと検査結果を提示します。
[バイタル・検査値を入力]
主な鑑別診断と、各疾患を支持・否定する所見を整理してください。
```

### 用途4: 間違えた問題の解説を深める

```
【プロンプト例】
次の問題を間違えました。正答はBですが私はCを選びました。
なぜBが正解でCが誤りなのか、病態生理から詳しく説明してください。
[問題文を貼り付ける]
```

### 用途5: まとめノートを作る

```
【プロンプト例】
内科専門医試験で出やすい「電解質異常（低Na血症・高K血症）」の
重要ポイントを、表形式でまとめてください。
原因・症状・治療の3列で作成してください。
```

## ChatGPT vs Claude、どちらを使うか

| 用途 | ChatGPT (GPT-4o) | Claude |
|------|-----------------|--------|
| 問題作成 | ◎ | ○ |
| 医学的説明の深さ | ○ | ◎ |
| 長文の整理・要約 | ○ | ◎ |
| 最新の医学情報 | ○（Web検索あり） | △ |
| 日本語の自然さ | ○ | ◎ |

試験対策の質問にはClaudeが特に向いています。長い疾患の解説や鑑別診断の整理を依頼すると、体系的で読みやすいまとめを作ってくれます。

その他のAI活用法は[専攻医のAI活用完全ガイド](/blog/j01-josler-ai-katsuyo)も参照してください。

## AI活用の注意点

### 医学的正確性を必ず確認する

AIは誤った医学情報を自信満々に提示することがあります（ハルシネーション）。AIの回答は**必ず教科書・ガイドラインで確認**する習慣をつけましょう。

特に数値（用量・検査基準値）は必ずダブルチェックが必要です。

### AIは補助ツールとして位置づける

AIはあくまで「補助ツール」です。理解の深掘り・問題演習の多様化に使い、メインの学習は教科書・問題集を軸にすることが重要です。

## よくある質問（FAQ）

**Q. AI生成の問題は実際の試験と同じレベルですか？**
A. 完全に同じではありませんが、考え方の練習として有効です。公式の過去問と組み合わせて使うことを推奨します。

**Q. AIに個人情報（患者情報）を入力してもよいですか？**
A. 絶対に入力しないでください。患者情報は匿名化・脱識別化した上でAIを使用してください。
